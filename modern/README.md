# Conceal Website - Modern

## Development

Run the development server:

```bash
npm run dev
```

## Build

Build for production:

```bash
npm run build
```

## Biome Scripts

### Linting

Check for linting issues:

```bash
npm run lint
```

Check and automatically fix linting issues:

```bash
npm run lint:fix
```

### Formatting

Check code formatting:

```bash
npm run format
```

Check and automatically fix code formatting:

```bash
npm run format:fix
```
