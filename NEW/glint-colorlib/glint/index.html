We Are Conceal

The Conceal project began as an experiment to create an alternative crypto-currency, that now encompasses an unique block reward model, short-term deposits, long-term investments, and secure encrypted messaging. With the aim of creating a secure store-of-value that provides returns for short and long-term holders, with a laser-focus on privacy.
